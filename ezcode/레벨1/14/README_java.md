## 문제 설명

두 양의 정수 a와 b가 주어질 때, a와 b를 이어붙인 두 가지 경우:

a 뒤에 b를 붙인 값

b 뒤에 a를 붙인 값
을 비교해 더 큰 값을 반환하세요.
두 값이 같다면 a 뒤에 b를 붙인 값을 반환합니다.

### 입력
- 정수 a, b (1 ≤ a, b < 10,000)

### 출력
- 이어붙인 결과 중 더 큰 값을 나타내는 정수
