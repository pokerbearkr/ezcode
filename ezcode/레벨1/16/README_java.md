## 문제 설명

정수 start와 문자열 cmd가 공백으로 구분되어 주어집니다. 문자열 cmd에는 'w', 'a', 's', 'd'로 이루어진 명령어들이 있으며,
각 문자를 왼쪽부터 차례대로 적용하면서 start의 값을 다음 규칙에 따라 변경하세요:

'w' → +1

's' → -1

'd' → +10

'a' → -10

모든 명령어를 처리한 후의 최종 정수값을 반환하세요.
