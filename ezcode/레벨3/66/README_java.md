## 문제 설명

삼각형의 세 변의 길이가 주어질 때 이 삼각형이 직각(right) 삼각형인지 , 예각(acute) 삼각형 인지 , 둔각(obtuse) 삼각형인지를 판별 하라.

### 입력

세 변의 길이 a,b,c 가 주어진다.(a,b,c 는 10000 이하의 자연수)
이루어질 수 없는 삼각형은 주어지지 않는다.

### 출력

right,acute,obtuse 중 하나를 출력한다.
